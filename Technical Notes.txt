...Pyrefly\pyrefly> npm run start
atau
...Pyrefly\pyrefly> npm start
waktu live server aktif, setiap kali kita save file bakal auto-compile dan langsung update di localhost kita
untuk end live server tinggal keyboard interrupt (Ctrl + C)
